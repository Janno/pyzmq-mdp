Welcome to ØMQ Majordomo Protocol Implementation’s documentation!¶

This package implements the ØMQ Majordomo Protocol as specified under http://rfc.zeromq.org/spec:7

Terms used¶

worker
independant process offering exactly one service in request/reply fashion.
client
independant process using a service in request/reply fashion.
broker
independant process routing messages from a client to a worker and back.
worker id
the ØMQ socket identity of the worker socket communiocating with the broker.
